# LineFollowerRT26

## OTA upload

Ovaj firmware podrzava upload preko WiFi-ja preko ESP32 Arduino OTA mehanizma.

1. Povezi se na WiFi access point `Proton ETF`.
2. U Arduino IDE izaberi mrežni port za `LineFollower26`.
3. OTA lozinka je `PROTONETF3`.

Web stranica za PID/drive kontrolu i dalje radi na IP adresi AP-a.
